Features

> Hyprland WM

Dynamic tiling window manager with green-on-black hacker aesthetic. Blazing fast Wayland compositor with smooth animations.

> Omarchy-Style Menu

Press Super+Space to open the AnonOS Menu. Install packages, browse AUR, manage configs, and control your system - all from a rofi-powered interface.

> AUR + Pacman

Full Arch Linux repository access plus AUR support via yay. Categorized install menu for Editors, Browsers, Security tools, Gaming, and more.

> Hacker Theme

Matrix-green #00ff00 on deep black. Custom kitty terminal, waybar, rofi, dunst notifications, and hyprlock screen - all themed for the underground aesthetic.

> Security Toolkit

Pre-installed with nmap, Wireshark, Metasploit, John the Ripper, Hydra, sqlmap, aircrack-ng, and more. Ready for penetration testing out of the box.

> Developer Ready

Python, Node.js, Go, Rust, Docker, Neovim, Git - everything you need to build. Starship prompt, zoxide, fzf, bat, eza for a modern terminal experience.

Installation

root@anonos:~# curl -fsSL https://anonos.org/install | bash
# Or download the ISO and flash to USB:
root@anonos:~# wget https://github.com/karlthecoder105/AnonOS/releases/latest/download/anonos.iso
root@anonos:~# dd if=anonos.iso of=/dev/sdb bs=4M status=progress && sync
root@anonos:~# reboot

Minimum: 2GB RAM В· 20GB disk В· x86_64 В· UEFI or BIOS

Commands

  • anonos-menu вҖ” Open the main menu (or press Super+Space)
  • hi вҖ” Show system information
  • cm or matrix вҖ” Matrix rain effect
  • hack вҖ” Hollywood hacker mode
  • update вҖ” Full system update (pacman + yay)
  • install <pkg> вҖ” Install a package
  • search <term> вҖ” Search packages
  • aur <pkg> вҖ” Install from AUR
  • ports вҖ” Show open ports
  • myip вҖ” Show public IP
  • weather вҖ” Terminal weather
  • cheat <topic> вҖ” Cheat sheet for any topic
